Você precisará construir um sistema que tenha as seguintes funcionalidades gerais:
1- Login/Logout (para entregar, o usuário precisa se identificar) | login(),criptografarArquivo() e descriptografarArquivo()

2- Cadastro e remoção de usuário | cadastrarUsuario(), 	removerUsuario(), alterarNivelUsuario()

3- Cadastro de elementos | cadastrarCliente()

4- Remoção de elementos | removerCliente()

5- Busca de elementos | buscarCliente()

6- Atualização de elementos |alterarDadosCliente()

7- Ordenação de elementos | ordenarClientes()

8- Log das ações executadas no sistema |criarLogUsuario()

Os arquivos do sistema devem estar em um formato especial: | ordenarClientes()

9- Alguns arquivos deverão estar criptografados. |criptografarArquivo() e descriptografarArquivo(), criptografarElementos()

10 - Imprimir os elementos existentes em um arquivo sem criptografia. | mostruarioElementos()

11 - Modificação de nível de acesso de usuários | alterarNivelUsuario()
